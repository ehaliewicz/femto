femto - a tiny editor

building
----

gcc femto.c -lncurses -o femto


using
----

type anything you want or type Ctrl-x, then type s to save, or c to quit.


issues
----

- does not handle unicode
- does not handle resizing your terminal
- does not handle files larger than memory
